# Tetris Game in C++ and Qt

## 🇬🇧 English

This project consists of the design and development of a Tetris video game created as part of an engineering school project in Software Engineering and Object-Oriented Programming.

The objective of the project was to implement a functional version of Tetris using:
- C++
- Qt Framework
- UML modeling
- Object-oriented programming principles

The project includes a complete graphical user interface, several difficulty levels, a scoring system, tetromino management, collision detection, line clearing, and save/load functionality.

---

## 🚀 Features

- Graphical user interface using Qt
- Multiple difficulty levels
- Tetromino movement and rotation
- Collision detection
- Full line detection and removal
- Progressive score system
- Pause and resume system
- Save and continue functionality
- Tetromino projection system

---

## 🛠 Technologies Used

- C++
- Qt Creator
- UML
- Visual Paradigm
- Lucidchart

# Jeu Tetris en C++ et Qt
## 🇫🇷 Français

Ce projet consiste en la conception et au développement d’un jeu vidéo Tetris réalisé dans le cadre d’un projet scolaire de Génie Logiciel et de Programmation Orientée Objet.

L’objectif du projet était d’implémenter une version fonctionnelle du jeu en utilisant :

- le langage C++
- la bibliothèque graphique Qt
- la modélisation UML
- les principes de programmation orientée objet

Le projet comprend une interface graphique complète, plusieurs niveaux de difficulté, un système de score, la gestion des tétrominos, la détection des collisions, la suppression des lignes complètes ainsi qu’un système de sauvegarde et de reprise de partie.

## 🚀 Fonctionnalités
- Interface graphique avec Qt
- Plusieurs niveaux de difficulté

- Gestion des collisions
- Détection et suppression des lignes complètes
- Système de score évolutif
- Pause et reprise de partie
- Sauvegarde et chargement de partie
